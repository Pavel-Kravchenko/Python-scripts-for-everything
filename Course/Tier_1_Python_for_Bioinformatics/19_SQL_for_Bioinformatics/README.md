# 19. SQL for Bioinformatics

**Tier 1: Python for Bioinformatics**

Query biological databases using SQL. Learn SELECT, JOIN, and subqueries with Python's sqlite3 module — no server installation required.

## Topics Covered

- SELECT, WHERE, ORDER BY, GROUP BY
- Aggregate functions (COUNT, AVG, MAX, MIN, SUM)
- JOIN operations (INNER, LEFT, self-joins)
- Subqueries and set operations (UNION)
- CREATE TABLE, INSERT, UPDATE, DELETE
- Python sqlite3 integration with pandas
- Querying biological databases
