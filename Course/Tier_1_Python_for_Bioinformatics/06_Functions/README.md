# 1.06 Functions

**Tier 1: Python for Bioinformatics**

Defining and calling functions with all parameter types, lambda functions, scope rules, and recursion. Biological applications include GC content functions, sequence validators, and ORF finders.

## Topics Covered

- Defining functions with default arguments, `*args`, and `**kwargs`
- Lambda functions and `map`/`filter`/`reduce`
- Variable scope and the LEGB rule
- Recursive algorithms
- Docstrings and function documentation
- Biological applications: GC content, sequence validators, ORF finders
