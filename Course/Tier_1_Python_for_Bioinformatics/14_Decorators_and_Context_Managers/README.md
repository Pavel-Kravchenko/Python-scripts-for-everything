# 1.14 Decorators and Context Managers

**Tier 1: Python for Bioinformatics**

First-class functions and closures as the foundation for decorators, writing decorators from scratch, and context managers via `__enter__`/`__exit__` and `contextlib`. Applied to timing analysis functions, caching BLAST results, and managing database connections.

## Topics Covered

- First-class functions and closures
- Writing decorators with and without arguments
- `functools.wraps` for preserving function metadata
- Context managers with `__enter__`/`__exit__`
- The `contextlib` module
- Applied examples: timing sequence analysis, caching BLAST results, managing database connections
