# 1.17 Data Wrangling

**Tier 1: Python for Bioinformatics**

Cleaning and reshaping messy biological datasets: handling missing values, removing duplicates, pivoting between wide and long formats, and building readable transformation pipelines with method chaining.

## Topics Covered

- Handling missing values: when to drop vs. fill, biological context for each strategy
- Removing duplicates
- Data type conversion and cleaning messy annotation tables
- Reshaping between wide and long formats with `melt`, `pivot`, `stack`/`unstack`
- String operations for extracting information from annotation columns
- Custom transformations with `apply` and `applymap`
- Method chaining for clean, readable pipelines
