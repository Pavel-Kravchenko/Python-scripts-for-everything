# 1.07 File Operations

**Tier 1: Python for Bioinformatics**

Reading and writing text files with context managers, parsing FASTA files from scratch, and working with CSV, JSON, GenBank, and binary formats. Covers path handling and directory operations.

## Topics Covered

- Reading and writing files with the `with` statement
- File modes: read, write, append, binary
- Reading methods: `read()`, `readline()`, `readlines()`, iteration
- Parsing FASTA files: building a reader and writer from scratch
- CSV, JSON, Pickle, and GenBank formats
- Path handling and directory operations
