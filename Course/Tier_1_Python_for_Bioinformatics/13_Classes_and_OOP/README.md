# 1.13 Classes and OOP

**Tier 1: Python for Bioinformatics**

Object-oriented programming for modeling genes, sequences, and biological entities as objects. Covers classes, inheritance, polymorphism, dunder methods, and dataclasses for concise data modeling.

## Topics Covered

- Classes, `__init__` constructor, instance attributes and methods
- Dunder methods: `__str__`, `__repr__`, `__eq__`, `__lt__`
- Inheritance and polymorphism
- Abstract base classes
- Properties, class methods, and static methods
- Dataclasses for concise data modeling
