# 1.04 Strings and Sequences

**Tier 1: Python for Bioinformatics**

String methods for biological sequences: indexing, slicing, and immutability applied to DNA/RNA/protein alphabets. Covers reverse complement, transcription, codon extraction, motif finding, and GC content calculation.

## Topics Covered

- String indexing, slicing, and immutability
- DNA/RNA/protein alphabets as strings
- Reverse complement and transcription (DNA to RNA)
- Codon extraction
- Motif finding and pattern searching
- FASTA header parsing
- GC content calculation and f-string formatting for scientific output
