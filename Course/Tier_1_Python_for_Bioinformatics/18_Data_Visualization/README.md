# 1.18 Data Visualization

**Tier 1: Python for Bioinformatics**

Matplotlib fundamentals through publication-quality multi-panel figures, Seaborn for statistical plots, and biological plot types including volcano plots, MA plots, sequence logos, and genome coverage plots.

## Topics Covered

- Matplotlib figures, axes, and the object-oriented interface
- Plot types: line, scatter, bar, histogram, box, violin, heatmap
- Customization: colors, labels, legends, annotations, styles
- Seaborn: statistical plots, pair plots, clustered heatmaps
- Biological plot types: volcano plots, MA plots, sequence logos, genome coverage plots
- Multi-panel figures and saving publication-quality output
