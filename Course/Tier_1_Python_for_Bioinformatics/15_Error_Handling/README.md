# 1.15 Error Handling

**Tier 1: Python for Bioinformatics**

Structured error handling, custom exception classes, debugging techniques, and the `logging` module for production code. Includes handling messy bioinformatics data such as missing FASTA headers, ambiguous nucleotides, and network timeouts.

## Topics Covered

- `try`/`except`/`else`/`finally` structured error handling
- Raising exceptions and creating custom exception classes
- The Python exception hierarchy
- Debugging: print debugging, `assert`, `pdb`/`breakpoint()`
- The `logging` module for production bioinformatics code
- Handling messy data: missing FASTA headers, ambiguous nucleotides, NCBI network timeouts
