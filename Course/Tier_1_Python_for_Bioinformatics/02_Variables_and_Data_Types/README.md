# 1.02 Variables and Data Types

**Tier 1: Python for Bioinformatics**

Creating variables with Python naming conventions, all basic data types, type conversion, and f-strings for formatted output. Real bioinformatics calculations include molecular weights, GC content, and sequence length statistics.

## Topics Covered

- Variable creation and Python naming conventions
- Basic data types: `int`, `float`, `str`, `bool`, `None`
- Type conversion and type checking
- Using strings to represent biological sequences
- F-strings for formatted output
- Applied calculations: molecular weights, GC content, sequence length statistics
