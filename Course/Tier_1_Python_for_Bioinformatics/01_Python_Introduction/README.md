# 1.01 Python Introduction

**Tier 1: Python for Bioinformatics**

Why Python dominates bioinformatics, how to navigate Jupyter notebooks, and how to write your first bioinformatics program. Covers the Python ecosystem for biology and interactive exploration with the REPL.

## Topics Covered

- Why Python is the dominant language in bioinformatics
- Navigating Jupyter notebooks: editing cells, running code, interpreting output
- Writing your first bioinformatics program
- The Python biology ecosystem: BioPython, pandas, NumPy, matplotlib
- Installation and environment setup
- REPL basics and interactive exploration
