# 1.16 NumPy and Pandas

**Tier 1: Python for Bioinformatics**

NumPy fundamentals and Pandas DataFrames for tabular biological data. Applied to gene expression matrices, annotation tables, and experimental metadata with reading, filtering, grouping, and merging operations.

## Topics Covered

- NumPy: creating arrays, vectorized operations, broadcasting, fancy indexing, statistical functions
- Pandas Series and DataFrames
- Reading and writing CSV, TSV, and Excel files
- Indexing, selection, filtering, grouping, and aggregation
- Merging and joining datasets
- Applied to gene expression matrices, annotation tables, and experimental metadata
