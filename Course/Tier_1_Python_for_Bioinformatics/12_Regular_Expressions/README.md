# 1.12 Regular Expressions

**Tier 1: Python for Bioinformatics**

The `re` module from metacharacters to lookahead assertions, applied to restriction enzyme sites, NCBI ID extraction, PROSITE-to-regex conversion, and protein motif searching.

## Topics Covered

- `re` module: `search`, `match`, `findall`, `finditer`, `sub`, `compile`
- Metacharacters, character classes, quantifiers, groups, and backreferences
- Lookahead and lookbehind assertions
- Flags: `IGNORECASE`, `MULTILINE`, `DOTALL`, `VERBOSE`
- Bioinformatics applications: restriction enzyme sites, NCBI gene ID extraction, PROSITE patterns, protein motif searching, parsing structured biological text
