# 1.05 Control Flow

**Tier 1: Python for Bioinformatics**

Conditional statements and loops for iterating over sequences and datasets, with biological applications including classifying nucleotides and building a codon-to-amino-acid translator.

## Topics Covered

- `if`/`elif`/`else` for classifying biological data (purines vs. pyrimidines, amino acid properties)
- `for` and `while` loops for sequences and datasets
- Loop control: `break`, `continue`, and the `for...else` construct
- `enumerate()`, `zip()`, and `range()` usage
- Codon-to-amino-acid translation with a dictionary lookup inside a loop
