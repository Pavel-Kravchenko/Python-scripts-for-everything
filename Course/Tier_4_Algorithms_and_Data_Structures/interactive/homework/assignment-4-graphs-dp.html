Part A: Graph Algorithms (50 points)

Question 1 10 pts

Given the following graph, perform BFS and DFS starting from vertex A. List the order in which vertices are visited.

A --- B --- E | | | C --- D --- F

Adjacency List:

A: [B, C] B: [A, D, E] C: [A, D] D: [B, C, F] E: [B, F] F: [D, E]

Solution:

BFS (using queue): A → B → C → D → E → F

Process: Start A, enqueue B,C. Dequeue B, enqueue D,E. Dequeue C (D already visited). Dequeue D, enqueue F. Dequeue E, F (already queued). Dequeue F.


DFS (using stack/recursion): A → B → D → C → F → E

Or: A → C → D → B → E → F (depends on neighbor order)

Process: Visit A, go to B, go to D, go to C (backtrack from D), go to F, go to E.

Question 8 10 pts

Write the recurrence relation for the following problem:

Given an array of positive integers, find the maximum sum of non-adjacent elements.

Example: [3, 2, 5, 10, 7] → Answer: 15 (3 + 5 + 7)

Solution:

State Definition:

dp[i] = maximum sum of non-adjacent elements from index 0 to i


Recurrence Relation:

dp[i] = max(dp[i-1], dp[i-2] + nums[i]) Where: - dp[i-1] = skip current element - dp[i-2] + nums[i] = take current element

Base Cases:

  • dp[0] = nums[0]
  • dp[1] = max(nums[0], nums[1])

Example Trace for [3, 2, 5, 10, 7]:

  • dp[0] = 3
  • dp[1] = max(3, 2) = 3
  • dp[2] = max(3, 3+5) = 8
  • dp[3] = max(8, 3+10) = 13
  • dp[4] = max(13, 8+7) = 15 ✓
