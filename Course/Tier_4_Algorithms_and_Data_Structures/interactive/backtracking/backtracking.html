N-Queens Backtracking

def solve_n_queens(n):
    solutions = []
    board = [-1] * n  # board[row] = column of queen
    
    def is_safe(row, col):
        for r in range(row):
            c = board[r]
            if c == col:  # Same column
                return False
            if abs(r - row) == abs(c - col):  # Diagonal
                return False
        return True
    
    def backtrack(row):
        if row == n:
            solutions.append(board[:])
            return
        
        for col in range(n):
            if is_safe(row, col):
                board[row] = col
                backtrack(row + 1)
                board[row] = -1  # Backtrack
    
    backtrack(0)
    return solutions

Permutations via Backtracking

def permutations(arr):
    result = []
    n = len(arr)
    
    def backtrack(start):
        if start == n:
            result.append(arr[:])
            return
        
        for i in range(start, n):
            arr[start], arr[i] = arr[i], arr[start]  # Swap
            backtrack(start + 1)
            arr[start], arr[i] = arr[i], arr[start]  # Swap back
    
    backtrack(0)
    return result

What is Backtracking?

Backtracking is an algorithmic technique that explores all possible solutions by building candidates incrementally and abandoning ("backtracking") candidates that cannot lead to valid solutions.

Key Concepts

  • Choice: At each step, make a choice from available options
  • Constraint: Check if the choice satisfies all constraints
  • Goal: Check if we've reached a solution
  • Backtrack: Undo the choice and try the next option
