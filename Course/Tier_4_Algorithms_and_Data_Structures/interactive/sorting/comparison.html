💡 Key Insights

  • O(n²) vs O(n log n): Watch how the O(n log n) algorithms (Merge Sort, QuickSort) finish faster as array size increases.
  • Comparisons vs Swaps: Selection Sort has the fewest swaps but still many comparisons.
  • Constants Matter: For small arrays, simpler algorithms might be competitive despite worse Big-O.
  • QuickSort's Variance: QuickSort's performance depends on pivot selection. Bad pivots can make it O(n²)!
