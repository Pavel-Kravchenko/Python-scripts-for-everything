Algorithm Implementation

import heapq

def dijkstra(graph, source):
    # Initialize distances to infinity
    dist = {v: float('inf') for v in graph}
    dist[source] = 0
    prev = {v: None for v in graph}
    
    # Priority queue: (distance, vertex)
    pq = [(0, source)]
    visited = set()
    
    while pq:
        # Extract vertex with minimum distance
        d, u = heapq.heappop(pq)
        
        if u in visited:
            continue
        visited.add(u)
        
        # Relax all edges from u
        for v, weight in graph[u]:
            if v not in visited:
                new_dist = dist[u] + weight
                if new_dist < dist[v]:
                    dist[v] = new_dist
                    prev[v] = u
                    heapq.heappush(pq, (new_dist, v))
    
    return dist, prev

# Time: O((V + E) log V) with binary heap
# Space: O(V)

How Dijkstra's Algorithm Works

  1. Initialize: Set distance to source = 0, all others = ∞
  2. Add source to priority queue with distance 0
  3. While queue not empty:
    • Extract node with minimum distance
    • If already visited, skip
    • Mark as visited (finalized)
    • For each neighbor: if new path is shorter, update distance and add to queue
  4. Result: Shortest distances from source to all reachable nodes

Key Properties

  • ✅ Works with non-negative edge weights only
  • ✅ Greedy algorithm - always picks minimum distance next
  • ✅ Once a node is finalized, its distance is optimal
  • ⚠️ For negative weights, use Bellman-Ford instead
