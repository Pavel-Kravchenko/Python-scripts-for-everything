Edit Distance (Levenshtein)

Minimum operations to transform string 1 into string 2

📐 Recurrence Relation

if s1[i-1] == s2[j-1]:
  dp[i][j] = dp[i-1][j-1] // no operation needed
else:
  dp[i][j] = 1 + min(
    dp[i-1][j-1], // replace
    dp[i-1][j], // delete
    dp[i][j-1] // insert
  )
