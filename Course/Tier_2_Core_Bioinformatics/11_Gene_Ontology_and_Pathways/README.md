# 11. Gene Ontology and Pathways

**Tier 2: Core Bioinformatics**

Functional annotation of genes and proteins using the Gene Ontology system and KEGG pathway database. Covers GO enrichment analysis using statistical tests and interpretation of results in biological context.

## Topics Covered

- Gene Ontology: three ontologies (MF, BP, CC), DAG structure, GO terms, evidence codes
- GO enrichment analysis: hypergeometric test and Fisher's exact test
- KEGG database: pathway maps, enzymes, metabolic networks
- Functional annotation workflows
- Interpreting enrichment results: overrepresentation and gene set enrichment
