# 7. Protein Structure

**Tier 2: Core Bioinformatics**

Computational analysis of protein three-dimensional structure using BioPython's Bio.PDB module. Covers the PDB file format, structural hierarchy, geometric calculations, and 3D visualization in Jupyter.

## Topics Covered

- Four levels of protein structure: primary through quaternary
- PDB file format and ATOM record fields
- BioPython `Bio.PDB`: Structure/Model/Chain/Residue/Atom hierarchy
- Interatomic distances, bond angles, and dihedral angles
- Ramachandran plots
- RMSD calculation for structural comparison
- Secondary structure assignment with DSSP
- 3D visualization with py3Dmol
