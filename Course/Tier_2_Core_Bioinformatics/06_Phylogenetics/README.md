# 6. Phylogenetics

**Tier 2: Core Bioinformatics**

Building and interpreting phylogenetic trees from sequence data. Covers distance-based and character-based methods, tree construction with BioPython, and bootstrap analysis for evaluating branch support.

## Topics Covered

- Tree terminology: nodes, branches, clades, rooted vs. unrooted
- Distance-based methods: UPGMA and Neighbor-Joining
- Character-based methods: maximum parsimony, maximum likelihood
- Distance matrices from MSA
- Building and visualizing trees with BioPython's `Phylo` module
- Bootstrap analysis for branch support
- Applications: gene family evolution, viral tracking, species classification
