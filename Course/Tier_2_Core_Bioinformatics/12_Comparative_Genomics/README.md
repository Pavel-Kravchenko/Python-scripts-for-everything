# 12. Comparative Genomics

**Tier 2: Core Bioinformatics**

Identifying conserved elements and structural rearrangements by comparing genomes computationally. Implements dot plots from scratch and interprets visual patterns to study synteny, orthologs, and genome evolution.

## Topics Covered

- Goals and rationale of comparative genomics
- Implementing dot plots from scratch and interpreting visual patterns
- Diagonal lines (collinearity), breaks (rearrangements), parallel diagonals (repeats)
- Synteny analysis and conserved gene order
- Distinguishing orthologs from paralogs
- Proteome-level comparisons
