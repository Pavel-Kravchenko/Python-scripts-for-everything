# 5. Multiple Sequence Alignment

**Tier 2: Core Bioinformatics**

Why and how to align more than two sequences simultaneously to reveal conserved regions, functional domains, and evolutionary relationships across entire protein or gene families.

## Topics Covered

- Why MSA reveals what pairwise alignment cannot
- Computational challenge: NP-hardness and heuristic approaches
- Progressive alignment and guide tree construction
- Running ClustalW and MUSCLE from Python
- Conservation scoring and consensus sequences
- MSA visualization and interpretation
- Identifying conserved functional residues
