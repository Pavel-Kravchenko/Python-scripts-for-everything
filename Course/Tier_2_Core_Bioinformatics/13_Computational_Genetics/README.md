# 13. Computational Genetics

**Tier 2: Core Bioinformatics**

Decoding the genetic code, analyzing codon usage bias, and mapping genes computationally. Covers restriction enzyme digestion, open reading frame finding, genetic mapping from cross data, mutation spectrum analysis, and Hardy-Weinberg equilibrium testing — all implemented from scratch in Python.

## Topics Covered

- Building and using the standard genetic code; degeneracy and synonymous codons
- Relative Synonymous Codon Usage (RSCU) and the Codon Adaptation Index (CAI)
- Restriction enzyme recognition sites, virtual digest, and gel simulation
- Open reading frame detection across all 6 reading frames
- Genetic mapping: two-point and three-point crosses, coefficient of coincidence
- Transition/transversion ratios and mutation spectrum analysis
- Hardy-Weinberg equilibrium and chi-squared goodness-of-fit test
