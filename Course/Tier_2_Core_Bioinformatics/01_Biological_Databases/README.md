# 1. Biological Databases

**Tier 2: Core Bioinformatics**

A practical guide to navigating and querying the major bioinformatics databases programmatically. Covers the NCBI ecosystem, UniProt, and the Protein Data Bank, with hands-on use of BioPython's Entrez module.

## Topics Covered

- NCBI ecosystem: GenBank, RefSeq, Gene, SRA, GEO
- Entrez API and programmatic database access
- UniProt: Swiss-Prot vs. TrEMBL annotation tiers
- Protein Data Bank (PDB) for structural data
- Accession numbers, identifiers, and cross-references between databases
