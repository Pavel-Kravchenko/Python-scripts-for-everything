# 0. Skills Check: Core Bioinformatics

**Tier 2: Core Bioinformatics**

Self-assessment to determine whether you can skip Tier 2 and proceed directly to applied bioinformatics. Fifteen questions test BLAST, sequence alignment, phylogenetics, protein structure, and functional annotation.

## Topics Covered

- BLAST and E-value interpretation
- Substitution matrices (PAM, BLOSUM)
- Phylogenetic tree topology
- PDB format and protein structure
- GO enrichment analysis
- Bioinformatics glossary (108 terms, A-Z reference)
