# 3. Pairwise Sequence Alignment

**Tier 2: Core Bioinformatics**

The theory and implementation of pairwise sequence alignment, from substitution matrices to dynamic programming algorithms. Implements Needleman-Wunsch and Smith-Waterman from scratch, then uses BioPython's alignment modules.

## Topics Covered

- Dot plots as a visual alignment tool
- Substitution matrices: PAM and BLOSUM families, derivation, and selection
- Gap penalty schemes: linear vs. affine
- Needleman-Wunsch (global) algorithm with dynamic programming
- Smith-Waterman (local) algorithm with dynamic programming
- E-values and statistical significance
- BioPython `pairwise2` and `Align` modules
