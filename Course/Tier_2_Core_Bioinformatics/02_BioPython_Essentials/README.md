# 2. BioPython Essentials

**Tier 2: Core Bioinformatics**

Core BioPython objects and workflows for sequence analysis. Covers the Seq, SeqRecord, and SeqIO interfaces, and walks through a complete gene-to-protein analysis pipeline using NCBI data.

## Topics Covered

- `Seq` object: complement, reverse complement, transcription, translation
- `SeqRecord` for annotated sequences
- `SeqIO` for reading and writing FASTA, GenBank, EMBL formats
- Fetching sequences from NCBI programmatically with `Entrez`
- End-to-end workflow: fetch gene, find ORFs, translate, search for protein
