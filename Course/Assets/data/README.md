# Shared Data Files

Sample biological data files used by course notebooks. All files are small enough to keep in the repository.

## Sequence Files

- **Saccharomyces_cerevisiae.fasta** -- Yeast gene nucleotide sequence (754 B). Used in BioPython Essentials, Pairwise Sequence Alignment.
- **Bacillus_cereus.fasta** -- Bacterial gene nucleotide sequence (302 B). Used in BioPython Essentials, Pairwise Sequence Alignment.
- **Saccharomyces_COX2.fasta** -- Yeast COX2 gene with multiple sequence entries for alignment (19 KB). Used in Multiple Sequence Alignment, Phylogenetics.
- **Mouse_Mus_cytb.fasta** -- Mouse cytochrome b sequences from multiple specimens (277 KB). Used in Phylogenetics, Comparative Genomics.
- **Escherichia_coli_proteome.fasta** -- Complete E. coli proteome, all protein sequences (1.9 MB). Used in BLAST Searching, Sequence Motifs and Domains.

## Structure Files

- **sample_protein.pdb** -- Small PDB structure file (12 KB). Used in Protein Structure, Molecular Modeling and Docking.
- **EBLOSUM62.txt** -- BLOSUM62 substitution matrix in EMBOSS format (2.1 KB). Used in Pairwise Sequence Alignment, BLAST Searching.

## Chromatogram Files

- **WS1945_COI_F_D11_WSBS-Seq-4-08-15.ab1** -- Forward-read Sanger sequencing chromatogram for COI barcoding gene (224 KB). Used in Chromatogram Analysis, Nucleic Acid Structure.
- **WS1945_COI_R_D12_WSBS-Seq-4-08-15.ab1** -- Reverse-read Sanger sequencing chromatogram for COI barcoding gene (224 KB). Used in Chromatogram Analysis, Nucleic Acid Structure.

## Annotation and Variant Files

- **sequence.gb** -- GenBank-format annotated genomic sequence (50 KB). Used in Biological Databases, BioPython Essentials.
- **snp.vcf** -- VCF file with SNP variant calls (31 KB). Used in Variant Calling and SNP Analysis, NGS Fundamentals.

## Machine Learning Datasets

- **agaricus-lepiota.csv** -- Mushroom classification dataset with 8,124 instances and 22 categorical attributes (edible/poisonous). Classic ML teaching dataset. Used in Machine Learning for Biology.
- **wine.csv** -- Wine cultivar classification dataset with 178 instances and 13 chemical analysis features. Used in Machine Learning for Biology.
- **parkinsons.csv** -- Parkinson's disease vocal measurement dataset with 195 instances and 22 features. Status column is the target (1=PD, 0=healthy). Used in Machine Learning for Biology.

## Graph Data

- **graph1.txt** -- Weighted undirected graph with 15 edges (vertices A-H). Used in Graph Algorithms.
- **graph2.txt** -- Weighted directed graph with 25 edges including negative weights (vertices A-P). Used in Graph Algorithms.
