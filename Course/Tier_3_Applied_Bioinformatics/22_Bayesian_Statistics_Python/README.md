# Module 22: Bayesian Statistics in Python

**Tier 3 — Applied Bioinformatics | Module 22**

Bayesian statistical modeling with PyMC, Bambi, and ArviZ — covering seven sub-sections from frequentist comparisons to advanced GLMMs.

## What You'll Learn
- Frequentist vs Bayesian inference: credible intervals vs confidence intervals
- Prior specification: informative, weakly informative, prior predictive checks
- Multiple regression: collinearity, VIF, coefficient interpretation
- Model comparison: LOO-CV and WAIC with ArviZ
- Linear mixed-effects models with random intercepts/slopes (Bambi)
- GLMs: Poisson, Negative Binomial, Zero-Inflated
- Advanced: GLMM, GAMs, Bayesian meta-analysis

## Prerequisites
- Module 06 (Statistics for Bioinformatics) — hypothesis testing, distributions
- Module 07 (Machine Learning) — regression concepts

## Data
Public palmerpenguins dataset (or synthetic equivalent); simulated ecological count data.

## Attribution
Statistical concepts based on Fränzi Korner-Nievergelt's Applied Statistics course (original R). Python implementation by course authors.
