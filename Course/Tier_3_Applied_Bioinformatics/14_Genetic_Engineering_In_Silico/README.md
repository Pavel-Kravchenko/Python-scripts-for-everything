# Module 14: Genetic Engineering In Silico

This module implements core molecular cloning and genome-editing workflows entirely in Python, covering restriction digestion, gel electrophoresis simulation, and compatible-end analysis.
Primer design algorithms are built from first principles—4+2 rule, salt-adjusted Tm, and nearest-neighbor thermodynamics—with automated primer growing, dimer checking, and RE-site addition for cloning.
CRISPR guide RNA design covers PAM identification for SpCas9, SaCas9, and Cas12a, guide scoring by GC content and position-specific rules, and off-target mismatch enumeration.
Codon optimization is implemented with real codon usage tables for E. coli, yeast, and human, including CAI calculation, rare-codon cluster avoidance, and domain-boundary preservation.
Gibson Assembly primer design closes the module, followed by four integrative exercises combining restriction cloning, CRISPR guide ranking, codon optimization, and multi-fragment assembly.
