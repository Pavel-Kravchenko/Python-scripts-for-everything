# Module 20: Spatial Transcriptomics

**Tier 3 — Applied Bioinformatics | Module 20**

Spatially-resolved gene expression analysis with Squidpy and Scanpy.

## What You'll Learn
- AnnData structure with spatial coordinates
- Quality control for spatial data
- Normalization and clustering in spatial context
- Spatial neighbor graph construction
- Spatially variable gene detection (Moran's I)
- Cell-type deconvolution concepts
- Visualization: expression overlays on tissue sections

## Prerequisites
- Module 03 (RNA-seq Analysis) — gene expression basics
- Module 12 (Modern Bioinformatics Workflows) — single-cell/Scanpy basics

## Data
Public 10x Visium mouse brain dataset (bundled with Squidpy via `sq.datasets.visium_hne_adata()`).
