# 5. Promoter and Regulatory Analysis

**Tier 3: Applied Bioinformatics**

Computational identification of regulatory elements in genomic sequences. Covers TATA box detection, CpG island analysis, transcription factor binding site prediction, and DNA methylation patterns.

## Topics Covered

- Gene regulation: promoters, enhancers, and silencers
- TATA box identification and consensus sequences
- CpG islands: definition, detection algorithms, role in gene regulation
- Transcription factor binding site (TFBS) prediction using PWMs
- Scanning genomic sequences for motif hits
- DNA methylation patterns and biological significance
- Integrating regulatory predictions with expression data
