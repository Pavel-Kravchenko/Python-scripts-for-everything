# 8. Capstone Project: From Sequence to Discovery

**Tier 3: Applied Bioinformatics**

An end-to-end integrative project analyzing unknown DNA sequences using all skills acquired across every tier. Designed to be completed in 8-12 hours across multiple sessions as a final readiness check for independent bioinformatics work.

## Topics Covered

- Sequence identification via BLAST
- Multiple sequence alignment and conservation analysis
- Phylogenetic tree construction and interpretation
- Protein structure analysis
- Statistical evaluation of results
- Drawing biological conclusions from integrated evidence

- All Tier 1 modules: Python for Bioinformatics
- All Tier 2 modules: Core Bioinformatics
- Tier 3 Modules 1-7: Applied Bioinformatics
