# Module 21: DNA Copy Number Analysis

**Tier 3 — Applied Bioinformatics | Module 21**

Copy number variation analysis from sequencing read depth.

## What You'll Learn
- CNV concepts: gains, losses, loss of heterozygosity
- Read depth normalization and log2 ratio computation
- CBS segmentation algorithm (conceptual implementation)
- Integer copy number state calling
- Gene-level CNV annotation (pyranges/pybedtools pattern)
- Genome-wide CN profile visualization

## Prerequisites
- Module 01 (NGS Fundamentals) — read alignment and depth
- Module 02 (Variant Calling and SNP Analysis) — VCF and variant pipeline basics

## Data
Simulated read depth data (notebook generates synthetic data). Real data: TCGA WGS (public via GDC portal).
