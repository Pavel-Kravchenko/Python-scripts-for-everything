# 6. Statistics for Bioinformatics

**Tier 3: Applied Bioinformatics**

Statistical methods used throughout bioinformatics, with emphasis on multiple testing correction, survival analysis, and power analysis. Provides practical guidance for choosing the right test for biological data.

## Topics Covered

- Distributions in biology: normal, Poisson, negative binomial
- Hypothesis testing: t-tests, chi-squared, Mann-Whitney, Kruskal-Wallis
- Multiple testing correction: Bonferroni, Benjamini-Hochberg FDR, q-values
- Survival analysis: Kaplan-Meier curves and log-rank test
- Power analysis and sample size calculations
- Resampling methods: permutation tests and bootstrapping
