# Module 19: Genome-Wide Association Studies (GWAS)

**Tier 3 — Applied Bioinformatics | Module 19**

GWAS from study design to fine-mapping.

## What You'll Learn
- SNP and sample quality control
- Population stratification via PCA
- Per-SNP association testing (logistic regression)
- Manhattan and QQ plots
- LD and clumping
- Fine-mapping concepts (SuSiE credible sets)
- GWAS Catalog queries

## Prerequisites
- Module 02 (Variant Calling and SNP Analysis)
- Module 15 (Population Genetics and Molecular Evolution)
- Module 06 (Statistics for Bioinformatics) — multiple testing

## Data
1000 Genomes Project public VCF or simulated genotype data (notebook uses simulation).
