# 10. Deep Learning for Biology

**Tier 3: Applied Bioinformatics**

Neural network architectures applied to biological sequence, structure, and single-cell data. Covers CNNs for genomics, transformers and protein language models, AlphaFold, and variational autoencoders, with practical implementation in PyTorch.

## Topics Covered

- When deep learning outperforms classical ML
- Neural network fundamentals: perceptrons, activation functions, backpropagation
- CNNs for genomic sequences and protein structure prediction
- RNNs and LSTMs for sequential biological data
- Transformer architectures and attention mechanisms
- Protein language models: ESM and ProtTrans
- AlphaFold and the protein structure prediction revolution
- Variational autoencoders for single-cell data
