# 4. Microbial Diversity

**Tier 3: Applied Bioinformatics**

Amplicon sequencing analysis for profiling microbial communities. Covers the 16S rRNA workflow from environmental sample to OTU/ASV tables, diversity metrics, ordination, and statistical testing.

## Topics Covered

- 16S rRNA gene as a universal bacterial/archaeal marker
- Amplicon sequencing workflow: denoising with DADA2 vs. OTU clustering
- Taxonomic classification against SILVA, Greengenes, and UNITE
- Alpha diversity: observed species, Shannon, Simpson, Chao1
- Beta diversity: Bray-Curtis and UniFrac distances
- Ordination and visualization: PCoA and NMDS
- Statistical testing: PERMANOVA and ANOSIM; rarefaction curves
