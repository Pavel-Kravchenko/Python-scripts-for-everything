# 0.04 Character Encodings

**Tier 0: Computational Foundations**

From bits to bytes: binary, decimal, and hexadecimal representations, ASCII as the universal foundation, and UTF-8 encoding for cross-platform collaboration. Includes practical debugging of garbled text in bioinformatics data formats.

## Topics Covered

- Binary, decimal, and hexadecimal representations
- ASCII as the universal character encoding foundation
- UTF-8 and why it matters when sharing files across operating systems
- Binary vs. text file modes in Python
- Encoding issues in FASTA, GenBank, GFF, and PDB formats
- Debugging garbled text and encoding errors
