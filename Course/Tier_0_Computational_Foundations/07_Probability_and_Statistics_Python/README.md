# 0.07 Probability and Statistics with Python

**Tier 0: Computational Foundations**

Hands-on implementation of probability distributions, hypothesis testing, correlation, regression, and resampling methods using scipy.stats, numpy, pandas, and statsmodels — with synthetic biological datasets throughout.

## Topics Covered

- Probability distributions: normal, binomial, Poisson, negative binomial via scipy.stats unified API
- Descriptive statistics, histograms, KDE plots, box plots, and Q-Q plots
- Hypothesis testing: t-tests, Mann-Whitney U, chi-squared, Fisher's exact, Shapiro-Wilk, Levene's
- Multiple group comparisons: one-way ANOVA, Kruskal-Wallis, Tukey HSD post-hoc
- Correlation (Pearson, Spearman) and regression (simple and multiple) with diagnostics
- Bootstrap confidence intervals and permutation tests
- Power analysis and sample size determination
