# 0.03 Bash Scripting

**Tier 0: Computational Foundations**

Writing and executing Bash scripts to automate repetitive bioinformatics tasks, including batch processing of FASTQ, BAM, and VCF files and building reusable pipeline scripts.

## Topics Covered

- Variables, conditionals, for/while loops, and functions
- Batch processing of biological data files
- Robust scripts with `set -euo pipefail` and error handling
- Text processing with `awk`, `sed`, and `cut` inside scripts
- Building reusable analysis pipeline scripts
