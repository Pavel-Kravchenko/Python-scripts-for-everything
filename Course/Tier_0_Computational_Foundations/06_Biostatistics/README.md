# 0.06 Biostatistics Fundamentals

**Tier 0: Computational Foundations**

Populations vs. samples, probability distributions relevant to biology, the hypothesis testing framework, and multiple testing correction — with practical examples from differential expression, variant calling, and enrichment analysis.

## Topics Covered

- Types of data: nominal, ordinal, discrete, continuous
- Probability distributions: normal, Poisson, negative binomial, binomial
- Hypothesis testing: null/alternative hypotheses, test statistics, p-values
- Significance levels and interpretation
- Multiple testing correction: Bonferroni and Benjamini-Hochberg FDR
- Applications in differential expression, variant calling, and enrichment analysis
