# 0.01 Linux Fundamentals

**Tier 0: Computational Foundations**

File system navigation, pipes and redirections, and command-line tools for data-processing pipelines. Every example uses bioinformatics scenarios such as FASTA files, BAM processing, and FASTQ quality checks.

## Topics Covered

- File system navigation, creating, moving, and copying files
- Pipes and redirections for data pipelines
- `grep`, `find`, `awk`, `sed` for text processing
- Compressed file handling: gzip and tar
- Process management
- SSH connections to remote servers
