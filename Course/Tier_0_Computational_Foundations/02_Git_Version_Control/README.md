# 0.02 Git Version Control

**Tier 0: Computational Foundations**

Why version control matters for reproducible research, covering everything from initializing a repository to collaborating on GitHub with branches and pull requests.

## Topics Covered

- Initializing repositories, staging, and committing changes
- Viewing history with `log`, `diff`, and `show`
- Branching and merging for feature development
- Collaborating with GitHub: push, pull, pull requests
- Writing `.gitignore` files for bioinformatics projects
- Undoing mistakes safely with `reset` and `revert`
