# 0.05 R Fundamentals

**Tier 0: Computational Foundations**

Enough R literacy to read R-based methods papers and run Bioconductor packages when needed, covering R basics through vectorized operations, data frames, and plotting.

## Topics Covered

- R assignment, types, vectors, and indexing (1-based, exclusion by negative index)
- Vectorized operations as R's core strength
- Data frames and factors
- Basic plotting
- The Bioconductor ecosystem: DESeq2, edgeR, Seurat
- ggplot2 fundamentals
