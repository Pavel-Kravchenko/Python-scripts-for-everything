# Tier 3: Applied Bioinformatics - Assignments

## Assignment Sets

### Set 1: NGS Pipelines (Modules 1-4)
Covers: NGS fundamentals, variant calling, RNA-seq analysis, microbial diversity

### Set 2: Analysis and ML (Modules 5-11)
Covers: Promoter analysis, statistics, machine learning, deep learning, clinical genomics

## How to Use
1. Work through the assignment notebook
2. Check your answers against the solution notebook in `Solutions/`
3. Each problem has a difficulty rating and grading rubric

## Grading Criteria

- **Correctness** (40%): Does the code produce correct results?
- **Code Quality** (30%): Is the code readable, well-organized?
- **Error Handling** (20%): Does it handle edge cases?
- **Documentation** (10%): Are functions documented?

## Difficulty Ratings

- (1 star): Beginner — direct implementation, single-step logic
- (2 stars): Intermediate — algorithm implementation, multi-step processing
- (3 stars): Advanced — complex algorithms, statistical methods
