# Tier 4: Algorithms & Data Structures - Assignments

## Assignment Sets

### Set 1: Foundations (Modules 1-6)
Covers: Complexity analysis, sorting, searching, linear data structures, trees, hash-based structures

### Set 2: Strings and Trees (Modules 7-8)
Covers: String matching algorithms (KMP, Rabin-Karp), tries, suffix arrays, suffix trees

### Set 3: Graphs and Dynamic Programming (Modules 9-10)
Covers: Graph traversal, shortest paths, MST, topological sort, memoization, tabulation, knapsack, alignment
