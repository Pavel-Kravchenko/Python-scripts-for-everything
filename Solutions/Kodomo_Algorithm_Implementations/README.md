# Kodomo Algorithm Implementations

Original algorithm implementations from the Kodomo Bioinformatics Program coursework at Lomonosov Moscow State University (5th semester, Algorithms & Data Structures).

These are cleaned-up versions of student solutions, preserved as reference implementations alongside the polished course material. They show alternative approaches and coding styles.

## Files

| File | Algorithm | Course Module |
|------|-----------|---------------|
| kmp.py | Knuth-Morris-Pratt | Tier 4, Module 07 |
| rabin_karp.py | Rabin-Karp rolling hash | Tier 4, Module 07 |
| dfa_matching.py | DFA pattern matching | Tier 4, Module 07 |
| naive_search.py | Naive string search + prefix function | Tier 4, Module 07 |
| aho_corasick.py | Aho-Corasick multi-pattern | Tier 4, Module 08 |
| hash_table_chaining.py | Hash table (chaining) | Tier 4, Module 06 |
| hash_table_open_addressing.py | Hash table (open addressing) | Tier 4, Module 06 |
| bloom_filter.py | Bloom filter | Tier 4, Module 06 |
| dynamic_array.py | Dynamic array | Tier 4, Module 04 |
